1. Jelaskan cara kerja Website serta jelaskan hubungan antara Web Server, Domain, dan Hosting !

Cara kerja website melibatkan beberapa komponen utama, termasuk web server, domain, dan hosting.
Cara kerja website melibatkan serangkaian langkah berikut:

1. Pengguna Memasukkan URL: Pengguna memasukkan alamat URL (Uniform Resource Locator) atau klik tautan ke dalam browser mereka.
2. DNS Resolution: Browser mencari alamat IP terkait dengan domain menggunakan sistem DNS (Domain Name System).
3. TCP Connection: Browser menginisiasi koneksi TCP (Transmission Control Protocol) ke server web yang terkait dengan alamat IP.
4. HTTP Request: Browser mengirimkan permintaan HTTP (Hypertext Transfer Protocol) ke server, meminta halaman web atau sumber daya lainnya.
5. Server Processing: Server menerima permintaan, memprosesnya, dan menghasilkan konten yang diperlukan, seperti HTML, CSS, dan JavaScript.
6. HTTP Response: Server mengirim kembali respons HTTP ke browser dengan konten yang diminta.
7. Rendering: Browser merender konten menggunakan HTML, CSS, dan JavaScript.
8. Display ke Pengguna: Halaman web ditampilkan kepada pengguna dalam browser.

Hubungan antara web server, domain, dan hosting


- Web server adalah perangkat keras atau perangkat lunak yang menyimpan, mengelola, dan mengirimkan konten web kepada pengguna.
- Domain berfungsi sebagai alamat yang memungkinkan pengguna mengakses situs web dengan mudah.
- Hosting menyediakan ruang server tempat file-file situs web disimpan sehingga dapat diakses oleh web server saat diperlukan.
